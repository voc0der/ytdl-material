# Security Policy

## Supported Versions

If you would like to see the latest updates, use the `nightly` tag on Docker.

If you'd like to stick with more stable releases, use the `latest` tag on Docker or download the [latest release here](https://github.com/voc0der/ytdl-material/releases/latest).

## Reporting a Vulnerability

Please file an issue in the GitHub repository.

ytdl-material is intended to be deployed as an internal-facing application and should not be treated as hardened for public exposure. It has not undergone formal penetration testing.

We welcome PRs and other contributions that improve ytdl-material's security posture.
